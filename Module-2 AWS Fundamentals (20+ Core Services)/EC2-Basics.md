## What is EC2?
- **EC2:** A service that provides resizable virtual servers in the cloud.

## Types of EC2 (Pricing Models)
- On-Demand
- Reserved
- Spot
- Dedicated Hosts
- Savings Plans

## What is an Image (AMI)?
- **AMI (Amazon Machine Image):** A preconfigured template containing an operating system, software, and settings used to launch an EC2 instance.

## EC2 Instance Types
- General Purpose
- Compute Optimized
- Memory Optimized
- Storage Optimized
- Accelerated Computing
